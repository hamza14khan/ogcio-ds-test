# ogcio-ds-test

OGCIO design system integration using vite.

## Tech

- [OGCIO/DS] - HTML enhanced for web apps!

## Installation

This requires [Node.js](https://nodejs.org/) to run.

Install the dependencies for server and frontend and run them respectively.

Frontend:

```sh
npm i
npm run dev
```
